# Cozinha 3D — Projeto Sogro

Esboço 3D navegável da cozinha planejada, em escala aproximada (3,00m x 3,50m, pé-direito 2,70m estimado).

Layout baseado no modelo escolhido na loja: bancada em L, ilha central em mármore com banquetas, coluna de forno + micro-ondas embutidos, geladeira inox, cooktop com coifa.

> ⚠ Medidas ainda não confirmadas pela medição oficial da loja — ajustar assim que a medição real for feita.

Feito com Three.js, arquivo único `index.html` — sem build, sem dependências além do CDN.
